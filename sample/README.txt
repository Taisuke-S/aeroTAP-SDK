=============================================================
	Ubuntu
=============================================================

Dependencies:

	sudo apt-get install freeglut3-dev

Dependencies for Ubuntu 14.04:

	sudo apt-get install libgl1-mesa-dri

Build sample:

	cd <path_to_AEROTAP-SDK_folder>/samples/aerotap_sample
	mkdir build
	cd build
	cmake ..
	make


Run sample:
	./aerotap_sample
